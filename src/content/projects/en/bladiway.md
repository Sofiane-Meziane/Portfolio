## Summary
A carpooling mobile app designed to connect drivers and passengers with a secure, smooth user experience. Includes real-time messaging, interactive maps, and an admin dashboard.

## Problem
Simplify shared rides with a trusted, multilingual mobile experience and a back-office interface for moderation and operations.

## Approach
- Flutter mobile app for riders and drivers.
- Firebase Authentication, Firestore, Storage, and Cloud Messaging.
- React admin dashboard for user management, document verification, and complaints.

## Results
Delivered a full functional prototype covering authentication, ride booking, messaging, notifications, and reviews.
